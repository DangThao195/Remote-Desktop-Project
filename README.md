# Remote Desktop with AI

**Project Description:**
• Developed a smart remote desktop application integrated with YOLOv8 for real-time user behavior monitoring.
The system analyzes camera feeds to assess user attention and detect entertainment-related distractions,
ensuring focused interaction while maintaining high system stability and low-latency communication.

**Responsibility**
• Designed and implemented client–server architecture for remote desktop communication
• Developed back-end logic for data transmission, connection handling, and system control
• Built front-end interfaces for user interaction and remote monitoring
• Integrated basic AI concepts for intelligent processing and analysis
• Collaborated in project planning, documentation, and version control using Git

**Tools Used**
• Programming Languages: Python
• Technologies: Socket Programming, Client–Server Architecture
• AI/ML: YOLO8
• Front-end: PyQT6
• Version Control: Git, GitHub
• Development Tools: VS Code
